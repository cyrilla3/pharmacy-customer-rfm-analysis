# pharmacy-customer-rfm-analysis
## RFM-анализ клиентской базы аптечной сети
1. Предположим, что RFM-анализ делается для держателей бонусных карт (таблица bonuscheques), ведь только у них есть идентификатор для персонализации и только им можно отправить СМС, так как скорее всего номера их бонусных карт привязаны к номерам их телефонов. При этом, клиентов, у которых нет бонусной карты, мы потеряем (кол-во потерянных покупателей можно будет посмотреть, сравнив итоговую сумму продаж из таблицы sales и bonuscheques).
2. В качестве анализируемого периода выбрано полугодие (01.01.2022 – 09.06.2022). Этого срока достаточно, чтобы охватить разные паттерны покупательского поведения в аптеке, но стоит учитывать, что клиенты с циклом покупок более 6 месяцев могут быть ошибочно отнесены к "худшим".
