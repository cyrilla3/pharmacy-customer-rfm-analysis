# 💊 Pharmacy Customer RFM Analysis

## RFM-анализ клиентской базы аптечной сети

---

### 📌 О проекте

**Цель:** провести сегментацию клиентов на основе истории покупок для персонализированной СМС-рассылки перед сезонным спадом продаж.

**Метод:** RFM-анализ (Recency, Frequency, Monetary) с разбивкой на 3 группы.

**Результат:** классификация клиентов по 27 сегментам, приоритизация и рекомендации по коммуникации.

---

### 🗄️ Данные

**Источник:** БД Apteka, таблица `bonuscheques`

| Поле | Описание |
|------|----------|
| `datetime` | дата и время покупки |
| `card` | номер бонусной карты (идентификатор клиента) |
| `doc_id` | номер чека |
| `summ` | сумма чека |
| `bonus_earned` | начисленные бонусы |
| `bonus_spent` | потраченные бонусы |

**Период анализа:** 01.01.2022 – 09.06.2022

---

### 🧠 Методология

#### RFM-метрики
| Метрика | Формула |
|---------|---------|
| **Recency** | количество дней с последней покупки до 09.06.2022 |
| **Frequency** | количество уникальных чеков (`doc_id`) |
| **Monetary** | сумма всех покупок (`summ`) |

#### Группировка
- Каждая метрика разделена на 3 уровня (1–3)
- Границы: 33-й и 66-й перцентили
- Итоговый сегмент = `[R_score][F_score][M_score]` (например, 333, 121)

### 📌 Ограничения

- Анализируются только клиенты с бонусной картой (реальные номера, отфильтрованы зашифрованные)
- Клиенты без карты не включены в сегментацию
- Период в 6 месяцев может не учитывать клиентов с циклом покупок более полугода
